# StardewFarm
DRAW

<Unity 사용 버전>
- 2020.3.34f1

<진행 상황>
- 키보드 입력을 통한 플레이어 좌우동작 (WASD)
- 좌우동작하는 플레이어 애니메이션
- 캐릭터가 움직이면 카메라가 같이 움직임
- 충돌
- 맵
  - 플레이어의 마당
- 인벤토리
  - I 키를 누르면 인벤토리 열고 닫기함
  - 스크롤바 가능
  - 아이템 이미지 적용
  - 인벤토리 확장
- 아이템
  - 아이템 DB
    - 장비, 일부 소모품(포션, 농약?)
  - 포션 사용
    - 소모품이기에, (마우스 클릭)사용하면 인벤토리에서 아이템 사라지도록 구현함.
